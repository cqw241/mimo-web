项目一：
名称：睿察 · 心链
类型：心理学教授支持的科研项目
一句话介绍：面向高校心理支持场景的多智能体 AI 辅助识别系统。
详细描述：融合文本、语音与面部线索，通过 LangGraph Agent 编排、多模态分析与风险分级策略，支持心理风险早期识别、脱敏告警、人工复核与规范转介闭环。
技术标签：LangGraph, Multi-Agent, Multimodal AI, Risk Grading, RAG, Full-stack
当前状态：可运行多智能体流程，完整前后端，可进行一次性测试，暂未支持长期账号登录
重点亮点：LangGraph Agent 编排、风险分级策略、多模态融合
注意：表达上必须强调它是智能辅助和人工复核系统，不替代专业心理诊断。

项目二：
名称：RepoPilot Harness
类型：个人项目
一句话介绍：面向 AI Coding Agent 的工程化 Harness、CI 修复评测与治理平台。
详细描述：以 CI 修复为首个高价值场景，将真实 CI 失败转化为可复现、可评测、可审计、可持续改进的私有工程资产。
技术标签：AI Coding Agent, CI Repair, Evaluation Harness, Governance, Developer Tools
重点亮点：工程化评测、CI 修复场景、可审计与可持续改进

项目三：
名称：GeoTaskShield
类型：个人项目
一句话介绍：面向移动群智感知的隐私保护任务分配仿真系统。
详细描述：通过量化任务完成率、系统收益、公平性与隐私损失等指标，对比不同隐私机制与分配算法对效率与隐私权衡的影响。系统基于 C++20 与 Qt 构建，提供可视化仿真、批量实验分析、Markdown/CSV 报告导出及智能实验辅助。
技术标签：C++20, Qt, Privacy Computing, Simulation, Visualization, Markdown Report, CSV Export
重点亮点：隐私效率权衡、可视化仿真、可复现实验分析
